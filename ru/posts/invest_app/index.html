Основные характеристики

Через это приложение вы можете получить информацию о своих доходах от купленных акций в рублях. Это приложение использует Tinkoff API для подключения к вашему инвестиционному портфелю. Получить токен Тинькофф API - официальный сайт

После этого приложение собирает информацию обо всех имеющихся в вашем портфеле акциях. Он захватывает следующую информацию:

  1. Название акции (нужно ввести вручную)
  2. Реальная стоимость акции - последняя цена от Тинькофф
  3. Количество акций, которые вы купили
  4. Средняя стоимость доли

Таким образом, вся информация собирается через API. И вы можете рассчитать общий доход с любой доступной акции в рублях. Также вы можете измерить возможный доход от акций, изменив параметр реальной стоимости акций.

Основные характеристики

  1. Использует API-токен Тинькофф
  2. Возможность импорта/экспорта токена API
  3. Возможность шифрования/дешифрования файла токена API для безопасного использования
  4. Отображает названия ваших акций в раскрывающемся списке.
  5. Совместимость с российскими акциями

Как создать это приложение?

  • Преобразование файла дизайна из QT Designer (файл .ui) в файл python .py:
  pyuic5 InvestApp.ui -о InvestApp.py
  • Собрать исполняемый файл (.exe) из main.py:
  pip install auto-py-to-exe
  авто-py-to-exe

В этом приложении вам нужно будет указать расположение скрипта main.py, некоторые значки и изображения, если они у вас есть, и после этого вы можете преобразовать свое приложение в файл .exe.
