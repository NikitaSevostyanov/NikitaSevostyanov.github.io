Building a Dynamic Web App using PHP & MySQL
Coursera Март 2022

Этот курс позволит научиться создавать простое веб-приложение для создания заметок с использованием PHP и MySQL. PHP и MySQL позволяют разрабатывать мощные динамические веб-страницы и приложения. В настоящее время они являются одними из наиболее широко используемых технологий в мире для разработки серверных приложений. Они позволяют вам создавать динамический контент на вашей веб-странице и воплощать его в жизнь. Они также позволяют сохранять данные в серверной базе данных и использовать их по-разному. Со временем вы сможете использовать изученные концепции для создания более сложных и мощных веб-страниц и приложений с использованием PHP и MySQL.
