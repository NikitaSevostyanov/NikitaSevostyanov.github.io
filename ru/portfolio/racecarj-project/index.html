Проект RaceCarJ

Автономный мобильный автомобиль

Описание:

Этот проект основан на проекте RACECARJ, созданном в Массачусетском технологическом институте (MIT).

Цель этого проекта - разработать автономный мобильный автомобиль и сгенерировать код для реального мобильного автомобиля, который может двигаться в соответствии с командами оператора, избегая возможных препятствий (с использованием алгоритма обхождения препятствий) и алгоритма следования по пути. Также в этом проекте были разработаны и реализованы следующие алгоритмы: Алгоритм локализации Монте-Карло - для определения местоположения автомобиля на текущей территории на основе карты, полученной методом SLAM (Simulteneous Localization and Mapping), также реализовано управление джойстиком.
